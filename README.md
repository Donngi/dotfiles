## Local values

Some dotfiles can load local dotfiles in your local $HOME additionally.

| file               | purpose                                  |
| ------------------ | ---------------------------------------- |
| `.zlocal`          | zsh local config                         |
| `.gitconfig.local` | git local config like user name or email |
